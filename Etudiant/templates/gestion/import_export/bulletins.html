ℹ️ Informations sur les formats

📄

PDF individuel

Génère un fichier ZIP contenant un bulletin PDF séparé pour chaque étudiant de la classe.

• Un PDF par élève • Téléchargement ZIP
📑

PDF groupé

Génère un seul fichier PDF contenant tous les bulletins de la classe avec saut de page entre chaque élève.

• Un seul PDF • Tous les élèves
📊

Excel

Génère un fichier Excel avec les données des bulletins sous format tableur pour analyse et traitement.

• Format tableur • Données analysables

📋 Instructions d'utilisation

  1. 1
    Sélectionnez la classe pour laquelle vous souhaitez générer les bulletins
  2. 2
    Indiquez le semestre (exemples : S1, S2, T1, T2)
  3. 3
    Précisez l'année scolaire (format : 2024-2025)
  4. 4
    Choisissez le format d'export selon vos besoins
  5. 5
    Cliquez sur "Générer" et patientez pendant la génération

⚙️ Informations techniques

Taille des fichiers : Les PDF peuvent être volumineux selon le nombre d'élèves
Temps de génération : Quelques secondes à quelques minutes selon la classe
Navigateurs supportés : Chrome, Firefox, Safari, Edge (versions récentes)
