Instructions d'utilisation

Import d'étudiants :

Le fichier doit contenir les colonnes suivantes :
numero_etudiant, nom, prenom, date_naissance, sexe, adresse, telephone, email, classe_id.
Les formats acceptés sont Excel (.xlsx, .xls) ou CSV (.csv).

Import de matières :

Le fichier doit contenir les colonnes suivantes :
nom, code, coefficient, description, enseignant_id, actif.
Les formats acceptés sont Excel (.xlsx, .xls) ou CSV (.csv).

Import de classes :

Le fichier doit contenir les colonnes suivantes :
nom, niveau, annee_scolaire.
Les formats acceptés sont Excel (.xlsx, .xls) ou CSV (.csv).

Exigences du fichier :

  • Formats acceptés : Excel (.xlsx, .xls) ou CSV (.csv)
  • Taille maximale : 5 MB
  • La première ligne doit contenir les en-têtes de colonnes
  • Les noms des colonnes doivent correspondre exactement à ceux indiqués
